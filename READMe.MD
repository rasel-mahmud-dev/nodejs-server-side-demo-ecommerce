### This Is Node js Server side simple online Ecommerce Demo



##### This Website build with 

- Nodejs
- Expressjs
- mongoose
- ejs
- nodemailer
- connect-flash
- etc...



First Install fresh npm packages

````sh
npm install 
````



Then to listen node js server and mongodb server 

```sh
npm start
```





Create a account  and add some product
